There are many choices for DBs

1. RDBMS (Relational DB Management Systems)
	1. SQL (not free, Oracle)
	2. MySQL
	3. PostgreSQL (free)
	4. MariaDB (free)
2. Aggregate-Oriented DBMS
	1. Key-Value DBMS
		1. Redis
	2. Document DBMS
		1. MongoDB
	3. Column-Family DBMS
		1. HBase
3. Graph DBMS
	1. Neo4j

RDBMS is the most generic choice for a database. Out of them, I chose to learn the free one, which is PostgreSQL. The following document helps a lot: https://www.postgresql.org/docs/current/tutorial.html
